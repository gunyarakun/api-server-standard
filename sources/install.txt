Install
=======

1. cd $lang
2. ./install.sh
3. ./start.sh release

When you execute './start.sh debug', reload editted files automatically (if supported).

Ruby
----

Requires rvm for Ruby.
Requires MySQL client library.

Perl
----

Requires cpanminus for Perl.
Requires MySQL client library.

Java
----

Requires JDK 6+.
