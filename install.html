3. Install¶

  1. cd $lang
  2. ./install.sh
  3. ./start.sh release

When you execute ‘./start.sh debug’, reload editted files automatically (if supported).

3.1. Ruby¶

Requires rvm for Ruby. Requires MySQL client library.

3.2. Perl¶

Requires cpanminus for Perl. Requires MySQL client library.

3.3. Java¶

Requires JDK 6+.
